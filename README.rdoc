== README

To obtain this directory clone from GitHub URL:

git clone git@github.com:polyphant1/TypeOff.git

Configured to Ruby 4.0.

SQ-Lite3 should be installed as standard for Heroku compatibility. All default Gems given in the Gemfile.
To view your currently installed Ruby gems run 'gem list'



Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
